# 🌳 Árbol BST Empresa en C++

## 👤 Autor
- Toainga Toapanta Snaider Alexander

## 📌 Descripción
Este proyecto implementa un **Árbol Binario de Búsqueda (BST)** en C++ para gestionar empleados dentro de una empresa.

Cada empleado es almacenado como un nodo dentro del árbol, permitiendo organizar la información de forma jerárquica y eficiente.

## 🎯 Objetivo
Desarrollar un sistema que permita gestionar empleados utilizando una estructura de datos no lineal (BST), aplicando operaciones fundamentales como inserción, búsqueda y recorridos.

## ⚙️ Funcionalidades

El sistema permite:

- ✔ Insertar empleados  
- ✔ Buscar empleados por código  
- ✔ Mostrar la raíz del árbol  
- ✔ Recorrido **Inorden**  
- ✔ Recorrido **Preorden**  
- ✔ Recorrido **Postorden**  
- ✔ Calcular la altura del árbol  
- ✔ Mostrar nodos hoja  


## 🧠 Conceptos utilizados

- **Raíz:** Nodo principal del árbol  
- **Nodo interno:** Nodo que tiene hijos  
- **Nodo hoja:** Nodo sin hijos  
- **Nivel:** Posición dentro del árbol  
- **Altura:** Número máximo de niveles  

## ▶️ Ejecución del programa

### Compilar:
g++ main.cpp arbol.cpp -o arbol     

## 📊 Resultados

El sistema permite organizar los empleados de forma jerárquica y realizar búsquedas eficientes utilizando un Árbol Binario de Búsqueda.

## 📌 Conclusión

El uso de árboles binarios de búsqueda permite gestionar información de manera eficiente, facilitando operaciones como búsqueda, inserción y recorrido en estructuras de datos no lineales.
